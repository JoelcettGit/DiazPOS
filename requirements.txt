# =====================================
# Notas de Instalación
# =====================================

# 1. Instalación en Desarrollo:
#    pip install -r requirements.txt
#    pip install -r requirements-dev.txt  (si existe)

# 2. Crear requirements-dev.txt con herramientas de desarrollo:
#    black, pylint, mypy, pytest, pytest-cov

# 3. Versiones pinned (==):
#    - Garantizan reproducibilidad
#    - Requieren actualizaciones manuales
#    - Usar: pip install --upgrade-all para actualizar todo

# 4. Verificar compatibilidad:
#    pip check

# 5. Generar requirements.txt desde entorno:
#    pip freeze > requirements.txt
